# 개발자 교육 과정 중 진행한 프로젝트에서 개인적으로 추가 개발을 진행.

## 사용 기술 스택 목록
프로그래밍 언어 : JAVA, JSP, Servlet <br>
웹 표준 : HTML, CSS, JavaScript, jQuery, ajax, Bootstrap <br>
프레임워크 : SpringFramwork <br>
DBMS : Oracle <br>
ORM : MyBatis <br>
Server : Tomcat <br>
BuildTool : maven <br>
형상관리툴 : Git/Github <br>
